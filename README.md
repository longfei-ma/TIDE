

## Running TIDE

TIDE supports two tasks:

- **Future Link Prediction (FLP)**
- **Dynamic Node Classification (DNC)**

Below are the commands for running each task on the **FOOD** dataset.

For FLP:

```bash
bash train_tide_link.sh FOOD 2 70 0
```

For DNC:

```bash
bash train_tide_node.sh FOOD 2 70 0
```
